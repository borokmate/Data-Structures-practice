1. ##### Linked List



* Clear - Clears the list
* IsEmpty - Checks if the list is empty
* Add - Adds a new element at the end of the list
* Remove - Removes a given element
* Count - A parameter that holds the count of elements
* Indexing - Can index like with a normal list (list\[x])
* Insert - Inserts a given element at an index
* IndexOf - Returns the index of a given element
* LastIndexOf - Returns the last index where the element was found
* Max - The max value of the list
* Min - The min value of the list
* RemoveAt - Removes at an index
* Instantiate with an array - can do new LinkedList<int> {3, 6, 1} 
* AddRange - adds a range at the end of the list
* RemoveRange - Removes a count of elements from a given index
* InsertRange - Inserts a range of elements at a given index
* ToString - When printed out it lists the elements like in python: 3, 6, 1



All methods have been given summaries and ect in the code
